Welcome to qinling-flyspeech's documentation!
====================================

qinling-flyspeech是基于MindSpore开发的语音预训练模型以及下游语音识别模型的算法工具包的简称，是西北工业大学和华为在探索智能语音道路上的重要成果，并计划发布中文预训练大模型，中文名称为秦岭·翔语，本仓是秦岭·翔语项目中的所有模型的代码，软硬件采用全国产华为昇腾全栈，包含预训练模型及下游任务模型。
